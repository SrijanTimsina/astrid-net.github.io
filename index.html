What is Tarsier?

Tarsier is a community driven multichain tools aggregator. 'Tools aggregator multichain' means this software allows you to use different tools to track your cryptoassets across multiple blockchains. 'Community driven' means that all decisions on the progress of the project, all its functions and features are decided by the members of its community. Anyone can help grow the project by joining the Telegram channel and proposing new ideas. The software was called Tarsier because it indicates the name of a cute big-eyed monkey with excellent eyesight. The purpose of the software is precisely to give you an excellent 'view' of all your cryptoassets.

What technologies does Tarsier use?

Tarsier is extremely complex software, consisting of eight different modules and over 1100 lines of code. Its features are guaranteed by a hybrid technology that combines excellent performance with high efficiency, with high standards of privacy for the user. Therefore the main approaches used to retreive user data are Data Gathering and Smart Contract ABI Execution.

the Data Gathering technique includes all those technologies such as server requests or web browser automation that allow the extraction of data from different web platforms.
The ABI execution method is slightly different. Basically, through some particular API we connect to a node on one of the supported chains, and we interrogate it, obtaining the data requested by us.

This hybrid approach allows us to optimize various technical processes, which would otherwise become slow or ineffective. However, Tarsier's initial software architecture was designed by one mind, and as such it can have limitations. Therefore we hope an increase of devs in the Tarsier community with which to collaborate to solve any limitations and to be able to grant software as performing as possible.

How to use Tarsier?
Click on one of the section to know more.
Software Launch
Currently the Tarsier software works on Telegram, so you can easily use it by entering from this link. You can start the bot by typing the command /start. If this is your first time using it you will be asked to enter an Ethereum address (Warning: Be sure the address is valid, otherwise it will not allow you to enter it), and then it will show you the Command Menu. If you have used Tarsier before, you will simply be shown the Bot Command Menu. It consists of the following buttons:
•Wallets buttons whose functionality in the Wallets section
•Token Tracker button whose functions are detailed in the token tracking section
•Scam Alert button whose functions are detailed in the Scam Detection section
•Add/Delete Wallet button whose functions are detailed in the Wallet Handling section

At any time you are you can stop Tarsier operation simply by typing the command /end.
Token Tracker
If you click the Token Tracker button you will be shown a menu with the following buttons:

•Track New Token If you want to track a new Token click this button. To insert the new token use this syntax: chain_name/contract_address/+ or -and percentage. The available chains are the following:
eth (Ethereum), bsc (Binance Smart Chain), ftm (Fantom), matic (Polygon), xdai (xDai), heco (Holochain), okt (OKEx). The contract address can easily be found on the web. For example, if you want to be alerted when the Cake token (found on the Binance Smart Chain) increases its price by + 20%, you should write like this:
bsc/0x0e09fabb73bd3ade0a17ecc321fd13a19e81ce82/+20%.
When the token will reach the target price Tarsier will sent you a message. •Delete Token if you no longer need to track the price of a token you can click on this button to delete it. Just enter the token contract address.
Scam Detector
Scam Detector is a function that allows you to have an in-depth analysis of a Token. Through it you will have several elements that will allow you to understand if a Token is a potential scam. This is a list of the data you will receive:
•Overview: Contains some general information about the token, such as circulating supply, max supply, market cap and number of holders. Projects with a low market cap or with low number of holders are riskier.
•Holders: When a large percentage of tokens are owned by a few wallets it is not favorable. PLEASE NOTE Some holders are accompanied by a contract emoji. It could indicate that a contract is associated with that address (it could indicate a liquidity contract with a Dex or a blocked address cannot sell its tokens). Contracts that indicate liquidity generally contain the name of the Dex with which it is entered into.
•Contract Verified: Blockchain Explores verification allows users to check, verify and confirm the transactions taking place in the blockchain. A verified contract reduces financial fraud by ensuring that there are no compromised databases.
•Ownership: Indicates who is the owner of the token smart contract. When this is possessed by the token creator, it can allow the latter to obtain special privileges (for example, it could prevent everyone from selling the token except him). It is good practice that ownership is renounced.
•Smart Contract Analysis: A scanner analyzes the Solidity code in the smart contract and identifies some potential risks. PLEASE NOTE: Sometimes this scanner fails to read liquidity contracts and alerts you to the inability to sell the token, even if it is not true. As a precaution, try to buy and sell a small amount of it. More information about scamming can be found on this reddit
Wallet Handling
The first time you will be shown the main menu of Tarsier, the first button will be the one that will allow you to manage the assets of the wallet you have registered. By clicking on this button you can access two features:
•Token Balance allows you to see the balance of the tokens on the wallet
•Defi Assets allows you to see your DeFi investments on over 400 protocols More information on supported chains and protocols can be found in the Home>what is section.
The last two buttons of the menu (Add Wallet / Delete Wallet) allow you to add other wallets to be tracked in case you need them.
